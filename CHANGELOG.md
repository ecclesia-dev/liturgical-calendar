# Changelog

- Traditional Roman Calendar ICS for import to Apple/Google Calendar
- Calendar expanded to 2024-2050 (3037 events, 27 years)
- Rust API source with calendar engine, crates, and tests
- SPEC.md documentation
- Pre-push hook auto-install script and CI workflow
- LICENSE and security gitignore patterns

### Fixed
- Septuagesima naming corrected
- Class II/III formatting improvements
- README and LICENSE reconciled
- Domain updated from ecclesia.dev to ecclesiadev.com in SPEC.md
